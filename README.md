# vue-oreo
A mobile UI Components based on Vue2.x 



## 规范

- `components`中名字均以小写出现,多个单词以`-`隔开,如`cell-group`

- [相关描述](./REMARK.md)

# 目前支持组件样式


- [x] Layout(基础布局)
    - [x] Flex
    - [x] Grid
- [x] Form
    - [x] Button(按钮)
    - [x] Textarea(文本)
    - [x] Switch(开关)
- [x] Cell(列表)
- [ ] Feedback
    - [x] Popup
    - [x] Dialog(对话框)    
    - [x] Toast
    - [x] Actionsheet
- [ ] Navigation
    - [x] Navbar(导航条)
    - [ ] Tabbar(标签栏)
    - [ ] Tabs(标签页)
- [ ] Display
    - [ ] Result(结果页)
    - [ ] Share(分享)
    - [ ] Steps(步骤条)
    - [ ] Card(卡片)
