# vue-oreo
A mobile UI Components based on Vue2.x &amp; [Oreo-UI](https://github.com/zhounan007/oreo-ui)


# 目前支持组件样式


- [x] Layout(基础布局)
    - [x] Flex
    - [x] Grid
- [x] Form
    - [x] Button(按钮)
    - [x] Radio(单选)
    - [x] CheckBox(多选)
    - [x] Textarea(文本)
    - [x] Switch(开关)
- [x] List(列表)
- [ ] Feedback
    - [x] Modal
    - [x] Dialog(对话框)    
    - [x] Toast
    - [x] Loading
    - [ ] actionsheet
- [ ] Navigation
    - [x] Navbar(导航条)
    - [ ] Tabbar(标签栏)
    - [x] Tabs(标签页)
- [ ] Display
    - [ ] Result(结果页)
    - [ ] Share(分享)
    - [ ] Steps(步骤条)
    - [ ] Card(卡片)
